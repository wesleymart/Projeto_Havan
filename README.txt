Olá 
Para fazer a excução do programa você deverá ter devidamente instalado o Eclipse ou outro programa que complile arquivos .java, Crie um projeto Java, dentro dele um pacote com o nome "main", em seguida copie as entidades Cliente, GerenciadorDAO e Gerenciador para dentro deste pacote.
Após essa etapa copie o arquivo txt com o nome "Conversor.txt", que está na raiz do projeto no GitHub, para a pasta raiz do seu HD, como por exemplo "C:\conversor.txt".
Agora basta executar a entidade Gerenciador que tem a main para execução do projeto.
O programa tem a opção de fazer um filtro por nome, poderá usar o nome "Carlos" ou "João" para testar, se quiser também pode abrir o txt e escolher alguns do nomes já cadastrados.  

OBS: o arquivo "conversor" já tem varios dados registrados nele, o que irá permitir a execução de todas as opções do programa.

Obrigado.
