Para fazer a excução do programa você deverá ter devidamente instalado o Eclipse ou Netbeans, em seguida copie as entidades Cliente, GerenciadorDAO e Gerenciador
para dentro de um projeto java dentro de um pacote com o nome de "main".
Após essa etapa copie o arquivo txt com o nome "Conversor.txt" para a pasta raiz do seu HD, como por exemplo "C:\conversor.txt".
Agora basta executar a entidade Gerenciador que tem a main para execução do projeto.

OBS: o arquivo "conversor" já tem varios dados registrados nele, o que irá permitir a execução de todas as opções do programa.
